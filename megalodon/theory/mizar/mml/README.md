# MML-Parallel Basics in Megalodon

These files are original Megalodon developments whose theorem statements and
naming follow the corresponding articles of the Mizar Mathematical Library
(MML): `card_1`, `card_2`, `finset_1`, `schems_1`, `subset_1`, `zfmisc_1`.

They were written from scratch as support lemmas for the Ramsey and
probability-theory developments in this repository; the proofs are not
translations of MML proof scripts.
